# Moderation System

## /ban
- Ban a member
- Optional reason
- Cannot ban yourself
- Cannot ban the bot
- Role hierarchy protection
- DM user before banning (if possible)
- Send moderation log

## /unban
- Unban a user using User ID
- Reason support
- Log action

## /kick
- Kick member
- Optional reason
- Permission checks
- Role hierarchy checks
- Log action

## /timeout
- Timeout member
- Duration support:
1m
5m
10m
30m
1h
6h
12h
1d
3d
7d

- Reason support
- Remove timeout option
- Log action

## /warn
- Warn users
- Save warnings inside warnings.json
- Add warning reason
- Show total warnings
- Log action

## /warnings
- View all warnings of a member

## /removewarn
- Remove a specific warning

## /clearwarns
- Delete all warnings of a member


# Additional Moderation Commands

/purge
- Delete 1-100 messages

/slowmode
- Set channel slowmode

/lock
- Lock a channel

/unlock
- Unlock a channel

/nickname
- Change user nickname


# Logging System

Create moderation logs for:

- Ban
- Unban
- Kick
- Timeout
- Warn
- Message Delete
- Message Edit
- Member Join
- Member Leave
- Role Update

Allow admins to set the log channel using:

/setlog


# Permission System

Require proper Discord permissions:

- Administrator
- Ban Members
- Kick Members
- Moderate Members
- Manage Messages
- Manage Nicknames

Include:
- Missing permission messages
- User hierarchy protection
- Bot hierarchy protection
